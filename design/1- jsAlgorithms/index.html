Yvette

本文是在看 《学习JavaScript数据结构与算法》中，

1- for … of 循环迭代

for … of 可以用来对数组进行迭代

2- 使用@@iterator对象

@@iterator是一个属性，需要通过 Symbol.iterator 来访问.会返回一个iterator（迭代器）。

3- 数组的entries、keys 和 values 方法

entries、keys 和 values 这三中方法都是可以从数组中得到迭代器的方法。

3.1 entries

如果你需要直接获取iter中的值，你也可以使用 for…of来遍历迭代器

3.2 keys / values

keys与values方法返回包含数组索引的@@iterator

4 - Array.from

Array.from() 可以根据现有的数组创建一个新的数组，也可以传入一个回调函数，用于对数组的返回值进行处理

5 - Array.of

通过Array.of()来新建或复制一个新数组。

6 - fill

静态方法填充，将数组中的元素进行替换
